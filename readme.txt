1.mini-imagecrop 是一款利用HTML5 canvas为实现方式的在线裁剪图片软件。体积小巧，功能紧致。核心文件仅有5.18kb。GZIP压缩后不足1KB \\n
2.支持GIF,PNG,JPEG,WEBP四种常用图片裁剪。具体因浏览器支持而异 \\n
3.支持圆形裁剪 \\n
4.支持任意宽高调整，或固定宽高比例调整。 \\n
5.支持固定尺寸导出和实际裁剪尺寸导出 \\n
6.丰富的自定义配置。 \\n
7.支持IE10（含）以上浏览器 \\n
8.支持AMD,CMD,commonJS,ES6导入 \\n



用法：
const crop = new Crop({
    view:".view"
})

请参阅：https://github.com/gapkukb/imageCrop
